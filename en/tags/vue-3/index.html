Vue3变化
  1. 性能提升，运行速度是vue2.x的1.5倍左右
  2. 体积更小，按需编译体积比vue2.x要更小
  3. 类型推断，更好的支持Ts（typescript）这个也是趋势
  4. 高级给予，暴露了更底层的API和提供更先进的内置组件
  5. ★组合API (composition api) ，能够更好的组织逻辑，封装逻辑，复用逻辑
